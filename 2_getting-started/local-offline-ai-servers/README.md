# Local Offline AI Server Setup

This section explains how to run AI Dev Kit with local backends instead of cloud APIs.

Use this when you want:

- No API key dependency for core local workflows
- Better privacy (data stays on your machine or your own server)
- Lower recurring cost for development and testing

***

## Available local setup guides

- **Self-Hosting with Ollama**: local LLM server for chat and embeddings
- **Self-Hosting with LM Studio**: local GGUF model serving with OpenAI-compatible endpoints
- **Setting Up Sherpa-ONNX**: offline STT/TTS runtime configuration
