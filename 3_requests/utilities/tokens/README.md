# Tokens

Tokenization utilities for AI models.

## Overview

The Tokens utilities provide functionality for:
- **Tokenize**: Convert text into tokens for AI model processing
- **Count Tokens**: Calculate the number of tokens in a given text

## Key Features

- Token counting for cost estimation
- Multi-provider token counting support
- Text-to-token conversion
- Model-specific tokenization

## Use Cases

- Estimate API costs before making requests
- Validate input length against model limits
- Debug prompt token usage
- Optimize prompt engineering
