# Generation Hubs

AI-powered generation windows built into the Unity Editor, accessible under `Tools > AI Dev Kit > Playgrounds`.

| Hub | Generated Assets |
|-----|-----------------|
| **Script Generation** | C# scripts, MonoBehaviours, ScriptableObjects, UXML |
| **Shader Generation** | Standard Surface, Unlit, Image Effect, Compute, Ray Tracing shaders |
| **Image Generation** | Characters, backgrounds, icons, mesh textures |
| **Image Editing** | Upscaling, inpainting |
| **Audio Generation** | Speech, sound effects, transcription, voice changes, music |
| **Video Generation** | Videos from text or image prompts |
| **Text Generation** | Plain text, OCR |
