# Title Generation

Title generation creates a short, meaningful conversation title automatically.

This is useful for chat history UIs where users need to quickly identify past conversations.

## Typical options

- **Enabled**: on/off
- **Trigger After Message Count**: when to generate first title
- **Regenerate Until Message Count**: optional re-generation window
- **Model**: model used for title generation

## Recommendation

Use a lightweight model for title generation to keep latency and cost low.
