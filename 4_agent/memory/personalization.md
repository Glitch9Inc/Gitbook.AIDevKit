# Personalization

Personalization uses user-specific information to adapt tone, style, and output relevance.

## Common inputs

- User name and role
- Domain preferences
- Communication style preferences
- Persistent user goals or constraints

## Best practices

- Keep profile data minimal and relevant
- Avoid sensitive data unless necessary
- Combine personalization with RAG and summaries for stable long-term behavior
